body: |
 
 This vulnerability allows remote attackers to execute arbitrary code on affected installations of Samsung MagicINFO 9 Server. Authentication is required to exploit this vulnerability.
 
 The specific flaw exists within the ContentSaveServiceImpl class. The issue results from the lack of proper validation of a user-supplied filename prior to using it in file operations. An attacker can leverage this vulnerability to execute code in the context of SYSTEM.
